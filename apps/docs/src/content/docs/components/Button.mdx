## Props

| プロパティ | 説明 |
|---|---|
|`variant`| `c--button--{variant}` クラスを出力します。`fill`,`outline`の2種類のスタイルをこの例では用意しています。|
{/* |`layout`| Buttonを構成するレイアウト用コンポーネントを指定できます。デフォルトは`Lism`です。| */}
{/* |`icon`, `endIcon` | ボタンテキストの左右に表示するアイコンを指定できます。| */}
{/* |`hover`| デフォルトは`fade`が指定されています。| */}


{/* ## 注意点
Common Props の `w`, `h` ではなく、`width`, `height` を使ってください。 */}



## Usage

{/* ### Buttonのvariantスタイル */}
{/* (`variant`を指定しなければ`'fill'`になります。) */}
{/* カラー指定は`keycol`を使います。 */}

（`variant`を使わず、`c`,`bgc`,`bdc`で自由に指定してもOKです。）

### `Grid`でアイコンを端に寄せる

`layout={Grid}`を指定すると gridレイアウト（`.l--grid`クラス）に切り替えれます。  
これにより、アイコンを端に寄せるレイアウトを組むこともできます。

<Preview>
	<PreviewTitle>Preview</PreviewTitle>
	<PreviewArea resize p='20' isFlow='20'>
		<Button href='###' variant='fill' layout={Grid} gtc='1em 1fr 1em' ji='c' g='10' maxW='25rem'>
			<Icon icon='cart'/>
			<GridItem tag='span' gc='2'>Button</GridItem>
			<Icon icon={ArrowRightIcon}/>
		</Button>
	</PreviewArea>
	<PreviewCode slot='tab' label='JSX'>
		```jsx "layout={Grid}" "gtc='1em 1fr 1em'" "ji='c'" "gc='2'"
		<Button href='###' variant='fill' layout={Grid} gtc='1em 1fr 1em' ji='c' g='10' maxW='25rem'>
			<Icon icon='cart'/>
			<GridItem tag='span' gc='2'>Button</GridItem>
			<Icon icon={ArrowRightIcon}/>
		</Button>
		```
	</PreviewCode>
</Preview>


こういったレイアウトも頻繁に使うのであれば、`.c--button:where(.l--grid)`にあらかじめCSSを当てておくと便利です。

<PreviewTitle>追加CSS</PreviewTitle>
```css
.c--button:where(.l--grid) {
	grid-template-columns: 1em 1fr 1em;
	justify-items: center;
}
.c--button:where(.l--grid) > :not(.l--icon) {
	grid-column: 2;
}
```

## Opt-in

### `hov="reverse"`

ホバー時に、fill,outlineが入れ替わるような [`-hov`クラス](/css/hover)を追加する例を紹介します。


<PreviewTitle>次のようなCSSを追加してください。</PreviewTitle>
```scss
.-hov\:reverse {
	--hov-c: var(--bgc);
	--hov-bgc: var(--c);
	&:where(.c--button--fill) {
		--hov-bgc: transparent;
	}
	&:where(.c--button--outline) {
		--hov-c: var(--c-base);
	}
	@media (any-hover: hover) {
		&:hover {
			background-color: var(--hov-bgc) !important;
			color: var(--hov-c) !important;
		}
	}
}
```
